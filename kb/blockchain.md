# Linux 上构建 geth

	Centos 6.5升级Git版本到Git2.7.2
	https://www.jianshu.com/p/371fe20db80c

	yum install go

	git clone https://github.com/ethereum/go-ethereum.git
	cd go-ethereum
	git checkout tags/v1.8.1
	make geth
	mv build /mnt/bin/geth
